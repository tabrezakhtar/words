#### Completed by Tabrez Akhtar

To run the app:

`npm install`

`node index`

The output should be:

```
checkout> node index   
[ 'a', 'of', 'on' ]
[ 'e', 'ddd', 'aa' ]
[ "won't", 'wont' ]
```

Run unit tests:  
`npm test`

To see the code coverage:  
`npm run coverage`  
open `coverage/lcov-report/index.html` in the browser
